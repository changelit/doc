# Git 
### 常用命令
* 初始化仓库，查询仓库状态

	git init
	git status

* git配置

	git config --global alias
	
* 远程下载

	git clone http://github.com/changelit/doc.git
	
* 管理修改

	git add -A
	git commit -m ""
	git diff

* 版本回滚

	git log -oneline
	git reset -hard

* 分支管理

	git checkout
	git checkout -b
	git branch
	git branch -D
	git merge
